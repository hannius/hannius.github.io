生成k8s证书的三种方式

发表于 2018-08-21 | 分类于 K8s

根据官方文档，生成k8s秘钥证书及相关管理证书有三种方式，其本质都是通过openssl:

  • cfssl
  • easyrsa
  • openssl

官方文档：https://kubernetes.io/docs/concepts/cluster-administration/certificates/

cfssl方式

2.生成CA配置文件
3.生成CA签名配置文件
4.生成私钥和证书
5.创建一个用于生成API Server的密钥和证书的JSON配置文件
6.生成 kubernetes 证书和私钥
7.创建admin证书
8.创建kube-proxy证书
9.校验证书信息
10.拷贝证书
easyrsa方式
1.下载：
2.生成 kubernetes 证书和私钥
3.签发admin证书
openssl方式
1.生成ca
2.kubernetes证书和私钥
3.admin证书
